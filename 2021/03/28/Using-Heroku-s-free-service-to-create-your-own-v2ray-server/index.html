Using Heroku's free service to create your own v2ray server

Posted on 2021-03-28

How to use Heroku free service to build a V2Ray server?

In today’s world, “Internet Security“ and “Internet Freedom“ are very important. Many people start to use VPN protocols, but most of the services are paid. So, how do you have VPN services of “Internet Security“ and “Internet Freedom“ without paying? We can use Heroku’s free service to create a V2Ray computing server, and then use the Cloudflare CDN free service to speed up the connection and keep our “Internet Security“ and “Internet free“!

Warning

Heroku provides us with free services, so we should not abuse this benefit. This article only introduces such operations and methods of use, and it is not recommended for long-term use.

let’s start!

  1. Open Github shuairuan/v2ray-heroku project website (I made some changes based on the original author bclswl0827), browse the README.md below (Block) Explanation, click on [Deploy to Heroku] in the sub-heading “Mirror“, and you will enter the page of [Heroku Create New App].

This is normal).

Step 2: Sign up for Cloudflare Workers

  1. Open the Cloudflare registration website and follow the on-screen instructions to register.

  2. After the registration is complete, you will see the account homepage, click the [Workers] link button in the right column.

  3. The first time you click in may require a registration operation, please follow the instructions on the screen to register (if the screen is in English, traditional Chinese can be selected in the upper right corner of the website; if you cannot find Traditional Chinese in the language menu, reduce the website a bit , You will see).

After registration, click [Create Workers].

  1. In the website, you will see a “Script block“ on the left, delete all the code in it, and copy the code below, and then click [Save Post deployment].

  1. After the deployment is complete, go back to the Workers homepage and click on the project you just created. On the upper right, there is a [Rename]. Please rename it to the URL you want for the V2Ray server. After naming, you can find the URL under “Available for workers subdomain” on the left side.

Download Clash and import the configuration file

  1. Open the Clash for Windows download page, select the version that suits you, download and install it.

  2. Go to Telegram Channel and download Simplified Chinese version of Clash file (please follow the instructions to install)

  3. Open Clash, in the [General] function page, install “TAP Network Adapter“, turn on “Service Mode“, click “**UWP Application Network Loopback (click to enter Then click “Exempt All”->”Save Changes”) **”.

  4. Open the [Configuration] function page, click [Import], then select the configuration file you just created, and then from the list in the [Configuration] function page, select the just imported File, make the left side of this option appear green (as shown in the figure below).

  1. Go to the [Proxy] function page, click [Global] at the top of the window, and then click [V2Ray Connection], Clash will start the V2Ray VPN service!
  • Suggestions: It is recommended to upload the configuration file to the Github repo and directly enter Clash, so that as soon as the configuration file is updated, Clash will be automatically updated.
